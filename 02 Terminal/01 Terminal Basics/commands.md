

## Some Basic Commands

cd          - change directory
cd ~        - change directory to the user's home directory
cd [path]   - go to the folder indicated


pwd         - prints the working directory


ls                  - lists contents of directory
ls /home/josh       - lists the contents of the josh directory
ls -a               - all files (includes hidden files)
ls -l               - lists the details of files 
                      ex: permission string, user that owns it, size, etc.

ls -lh              - h for human readable file sizes
ls -al
ls -alh


mkdir               - makes a directory
rmdir               - removes empty directories
touch               - makes files
rm                  - removes files
rm -rf              - removes directories with files

man                 - opens the manual
echo                - prints to the terminal
history             - shows the history of commands you typed
history -c          - clears
history > filename  - stores the history in a file

# Redirection
\>                   - redirects the standard output
